### Properties

* `onClick` - click callback
* `label` - button text
* `disabled` - boolean properties

| propName | propType | defaultValue | isRequired |
| -------- | -------- | ------------ | ---------- |
| onClick  | func     | -            |            |
| label    | string   | -            | true       |
| disabled | boolean  | false        | false      |
